3.What to Include When Contacting Support

Including these details in the first message lets us answer in one round trip instead of several:

  • Device model — for example "iPhone 16 Pro Max" or "iPad Air".
  • iOS or iPadOS version — found under Settings → General → About → Software Version.
  • CueFlow version — shown in the application settings or App Store listing.
  • Feature in use — Scripts, Prompter, Picture in Picture, import, settings, or legal/support.
  • What happened — a short description of the behavior, plus what you expected to happen.
  • Steps to reproduce — when possible, a short sequence we can follow on our end.

Please avoid sending private scripts unless you want us to review that text as part of your support request.

4.Common Topics

Floating Captions Do Not Start

CueFlow uses Picture in Picture to display floating captions. PiP availability depends on iOS, device support, system settings, and the current app environment. If floating captions do not start:

  • test on a real iPhone that supports Picture in Picture;
  • avoid relying on Simulator for final PiP verification;
  • open CueFlow in full screen, select a script, then tap Start Floating Captions;
  • make sure Picture in Picture has not been disabled in system settings;
  • return to CueFlow and try again if another app has interrupted playback.

Imported Document Has No Readable Text

CueFlow imports readable text from TXT, PDF, DOCX, RTF, and ODT files you select. Some PDFs are scans or images without embedded text, and some documents may contain formatting without extractable text. If import does not create a useful script, try copying the text from the original document or using a TXT file.

Managing Scripts

Scripts are managed from the Scripts tab. Tap a row to edit, use the play button to open the script in Prompter, and use the delete button to remove a script after confirmation. Deleted scripts cannot be restored by CueFlow.

Adjusting Speed, Font, and Opacity

Open Prompter to adjust speed, font size, line gap, opacity, and mirror mode directly from the controls. Changes apply immediately in the preview and are used when floating captions start.

Using CueFlow With Camera or Video Apps

Start floating captions in CueFlow first, then switch to the camera, meeting, streaming, or video app you want to use. The other app controls video capture, publishing, upload, and platform rules. CueFlow only provides the teleprompter overlay.

Text Looks Too Large or Too Small

Use Font Size and Line Gap in Prompter Settings. For close-to-camera talking videos, start with a moderate font size and slower speed, then adjust until the current line is readable without large eye movement.

Subscriptions and Restore Purchases

CueFlow requires an active subscription to use app features. Available plans include weekly, monthly, semiannual, and annual subscriptions. The weekly plan includes a 3-day free trial when offered in the App Store purchase sheet. Use Settings → Restore to restore a previous purchase made with the same Apple ID.

5.Privacy and Data

CueFlow does not require an account. Scripts are stored locally on device. Imported TXT, PDF, DOCX, RTF, and ODT files are read locally after you select them. Floating captions are rendered locally for Picture in Picture. CueFlow does not upload your script text to Omniform Labs servers.

Practical note. Any video or screen content you create in another app is controlled by that app and by iOS. Review the privacy and video capture rules of the app you use for capturing or publishing.
