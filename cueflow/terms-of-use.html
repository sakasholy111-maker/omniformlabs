2.Service Description

CueFlow is a focused teleprompter and floating captions application for iPhone and iPad. It provides:

  • Script Library — create, edit, search, select, and delete scripts for creator videos, speeches, demos, and talking points.
  • File Import — import readable text from user-selected TXT, PDF, DOCX, RTF, and ODT files.
  • Prompter Preview — preview lyric-style scrolling captions before starting playback.
  • Floating Captions — use Picture in Picture to display teleprompter captions over other apps when supported by iOS and the device.
  • Reading Controls — adjust speed, font size, line spacing, opacity, and mirror mode.

Features may be added, removed, or changed in future versions to keep the application aligned with its teleprompter purpose.
